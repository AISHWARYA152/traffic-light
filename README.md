# Traffic Light 🚦  
---
## 📂 Description  
A simple interactive traffic light built using **HTML, CSS, and JavaScript**.  

When you click the buttons:  
🔴 **Start** → Red light turns on  
🟡 **Ready** → Yellow light turns on  
🟢 **Go** → Green light turns on
